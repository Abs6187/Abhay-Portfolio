Project Overview

A sophisticated computer vision project that detects helmets in images and videos using the YOLOv11 object detection algorithm. This system is designed for workplace safety monitoring and compliance verification.

The project demonstrates advanced computer vision techniques with real-time detection capabilities, accuracy metrics visualization, and comprehensive evaluation using confusion matrices. Built with Python, OpenCV, and YOLOv11 for high-performance helmet detection in various scenarios.

Project Updates & Progress

Latest Development: Vehicle Speed Estimation and Counting

Extended the computer vision capabilities with a new Vehicle Speed Estimation and Counting system. This advanced project builds upon the helmet detection foundation to track vehicles, estimate their speeds, and count traffic flow in real-time.

Key Features:

  • Real-time vehicle detection and tracking using YOLOv8
  • Speed estimation based on pixel displacement and calibration
  • Vehicle counting with directional flow analysis
  • Support for multiple vehicle classes (cars, trucks, motorcycles, buses)
  • Interactive polygon zone definition for region-of-interest monitoring
  • Comprehensive analytics with speed distribution and traffic patterns

Technologies Used:

YOLOv8 ByteTrack Supervision OpenCV Streamlit Python
